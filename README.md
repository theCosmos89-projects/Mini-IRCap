# Mini-IRCap v2.5
Cliente GUI de IRC en español para Windows, inspirado en el script para mIRC, IRcap. Desarrollado en VB.NET utilizando la librería SmartIRC4net de Meebey.

Aplicación ligera y ejecutable de 386 KB (no requiere instalación). Compatible con cualquier red de IRC, está optimizada para la red de Irc-Hispano, con soporte para conexión cifrada SSL, admite eventos comunes y comandos básicos, opciones de Operador ("@") en menú, decodificación de texto con formato decorativo, fuente personalizable, tamaño de texto ajustable y cuatro colores de fondo disponibles. Cuenta con numerosos detalles refinados que la convierten en una aplicación intuitiva, fácil de usar, funcional y, sobre todo, práctica.

------------------------------------------------------------

La librería SmartIRC4net (.dll) ha sido integrada en el ejecutable mediante ILRepack, lo que podría generar falsos positivos en algunos antivirus, especialmente en aquellos de dudosa reputación. Además, aunque el archivo está firmado, su navegador podría bloquear el acceso.
Si esto representa un inconveniente, también está disponible una versión base comprimida en "Mini-IRCap.2.5.rar", que incluye la DLL por separado.

Dado que carezco de reputación y soy un programador amateur, recomiendo analizar Mini-IRCap en [VirusTotal](https://www.virustotal.com/) antes de ejecutarlo, para mayor confianza y tranquilidad.
