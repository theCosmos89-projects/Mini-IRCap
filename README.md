# Mini-IRCap v2.3
Cliente GUI de IRC en español para Windows, inspirado en el script IRcap para mIRC. Desarrollado en VB.NET utilizando la librería SmartIRC4net de Meebey.

Aplicación ligera y ejecutable de 362 KB (no requiere instalación). Funciona bien en la mayoria de los chats de las redes IRC, pero está especializado en la red IRC-Hispano. Compatible con conexiones cifradas SSL, incluye manejo de eventos básicos, sin soporte para comandos (opciones basicas de Operador: KICK/BAN/OP/DEOP...en menu), decodificación de texto decorativo, tamaño de texto ajustable y cuatro colores disponibles. Simple, funcional y práctica.

Nota: La librería SmartIRC4net (.dll) se integró junto al ejecutable utilizando ILMerge, lo que podría generar un falso positivo en algunos antivirus o el bloqueo por parte del navegador. Para evitar este inconveniente, también se ofrece una versión comprimida en el archivo "Mini-IRCap.2.3.rar", que incluye la .dll por separado.
