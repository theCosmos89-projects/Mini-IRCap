# Mini-IRCap v2.5
Cliente GUI de IRC en español para Windows, inspirado en el script para mIRC, IRcap. Desarrollado en VB.NET utilizando la librería SmartIRC4net de Meebey.

Aplicación ligera y ejecutable de 384 KB (no requiere instalación). Compatible con cualquier red de IRC, esta optimizada para la red de Irc-Hispano, con soporte para conexión cifrada SSL, admite eventos comunes y comandos básicos, opciones de Operador ("@") en menú, decodificación de texto con formato decorativo, fuente personalizable, tamaño de texto ajustable y cuatro colores de fondo disponibles. Cuenta con numerosos detalles refinados que la convierten en una aplicación simple, funcional y, sobre todo, práctica.

*** Para su seguridad, les animo a analizar Mini-IRCap en [VirusTotal](https://www.virustotal.com/) antes de ejecutarlo. La librería SmartIRC4net (.dll) ha sido integrada en el archivo mediante ILRepack, por lo que existe la posiblidad de que algunos antivirus, especialmente los de dudosa reputación, generen falsos positivos. Si esto representa un inconveniente, también se ofrece la versión base comprimida en "Mini-IRCap.2.5.rar", que incluye la DLL por separado.
