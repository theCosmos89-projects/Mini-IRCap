# Mini-IRCap v2.4
Cliente GUI de IRC en español para Windows, inspirado en el script IRcap para mIRC, especializado en la red de IRC-Hispano. Desarrollado en VB.NET utilizando la librería SmartIRC4net de Meebey.

Aplicación ligera y ejecutable de 371 KB (no requiere instalación). Compatible con conexiones cifradas SSL, incluye manejo de eventos básicos (opciones de Operador: KICK/BAN/OP/DEOP/INVITE...en menu), decodificación de texto decorativo, tamaño de texto ajustable y cuatro colores disponibles. Simple, funcional y práctica.

Nota: La librería SmartIRC4net (.dll) se integró junto al ejecutable utilizando ILMerge, lo que podría generar un falso positivo en algunos antivirus o el bloqueo por parte del navegador. Para evitar este inconveniente, también se ofrece una versión comprimida en el archivo "Mini-IRCap.2.4.rar", que incluye la .dll por separado.
