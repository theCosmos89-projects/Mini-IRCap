# Mini-IRCap v2.5
Cliente GUI de IRC en español para Windows, inspirado en el script IRcap para mIRC. Desarrollado en VB.NET utilizando la librería SmartIRC4net de Meebey.

Aplicación ligera y ejecutable de 384 KB (no requiere instalación). Compatible con cualquier red de IRC, esta optimizada para la red de Irc-Hispano, con soporte para conexión cifrada SSL, admite eventos comunes y comandos básicos, opciones de Operador ("@") en menú, decodificación de texto con formato decorativo, fuente personalizable, tamaño de texto ajustable y cuatro colores de fondo disponibles. Cuenta con numerosos detalles refinados que la convierten en una aplicación simple, funcional y, sobre todo, práctica.

Nota: La librería SmartIRC4net (.dll) se integró junto al ejecutable utilizando ILRepack, lo que podría generar un falso positivo y/o el bloqueo por parte de algunos antivirus o navegadores. Para evitar este inconveniente, también se ofrece una versión comprimida en el archivo "Mini-IRCap.2.5.rar", que incluye la dll por separado.
