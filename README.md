# Mini-IRCap v2.4
Cliente GUI de IRC en español para Windows, inspirado en el script IRcap para mIRC. Desarrollado en VB.NET utilizando la librería SmartIRC4net de Meebey.

Aplicación ligera y ejecutable de 350 KB (no requiere instalación). Compatible con cualquier red de IRC, esta optimizada para la red de IRC-Hipano con soporte para comandos y conexión cifrada SSL. Incluye manejo de eventos básicos, opciones de Operador (KICK, BAN, OP, DEOP, INVITE, etc., en menú), decodificación de texto decorativo, tamaño de texto ajustable y cuatro colores disponibles. Simple, funcional y práctica.

Nota: La librería SmartIRC4net (.dll) se integró junto al ejecutable utilizando ILRepack, lo que podría generar un falso positivo y/o el bloqueo por parte de algunos antivirus o navegadores. Para evitar este inconveniente, también se ofrece una versión comprimida en el archivo "Mini-IRCap.2.4.rar", que incluye la dll por separado.

IRC GUI Client for Windows in Spanish, inspired by the IRcap script for mIRC. Developed in VB.NET using Meebey's SmartIRC4net library.

A lightweight 350 KB executable application (no installation required). Compatible with any IRC network, optimized for the IRC-Hispano network, featuring support for commands and SSL-encrypted connections. Includes basic event handling, Operator options (KICK, BAN, OP, DEOP, INVITE, etc., via menu), decorative text decoding, adjustable text size, and four color schemes. Simple, functional, and practical.

Note: The SmartIRC4net library (.dll) was merged with the executable using ILRepack, which might trigger false positives and/or be blocked by certain antivirus software or browsers. To address this, a compressed version is also available in the "Mini-IRCap.2.4.rar" archive, including the dll separately.
