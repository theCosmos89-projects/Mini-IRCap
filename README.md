# Mini-IRCap v2.2
Cliente GUI de IRC en español para Windows, inspirado en el script IRcap para mIRC. Funciona bien en la mayoria de los chats de las redes IRC, pero está especializado en la red IRC-Hispano. Desarrollado en VB.NET utilizando la librería SmartIRC4net de Meebey.

Aplicación ligera y ejecutable de 344 KB (no requiere instalación). Compatible con conexiones cifradas SSL, incluye manejo de eventos básicos, sin soporte para comandos, decodificación de texto decorativo, tamaño de texto ajustable y cuatro colores disponibles. Está destinada a usuarios que solo desean chatear de forma cómoda sin administrar canales. Simple, funcional y práctica.

Nota: La librería SmartIRC4net (.dll) se integró junto al ejecutable utilizando ILMerge, lo que podría generar un falso positivo en algunos antivirus o el bloqueo por parte del navegador. Para evitar este inconveniente, también se ofrece una versión comprimida en el archivo "Mini-IRCap.2.2.rar", que incluye la .dll por separado.

---------------------------

Spanish IRC GUI Client for Windows, inspired by the IRcap script for mIRC. It works well in most IRC network chats but is specialized in the IRC-Hispano network. Developed in VB.NET using the SmartIRC4net library by Meebey.

A lightweight and executable application of 344 KB (no installation required). Compatible with SSL-encrypted connections, it includes basic event handling, does not support commands, decorative text decoding, adjustable text size, and four available colors. It is intended for users who only want to chat comfortably without managing channels. Simple, functional, and practical.

Note: The SmartIRC4net library (.dll) was integrated with the executable using ILMerge, which may trigger a false positive in some antivirus programs or cause browser block. To avoid this issue, a compressed version is also available in the "Mini-IRCap.2.2.rar" file, which includes the .dll separately.
