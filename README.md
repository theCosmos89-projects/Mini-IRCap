# Mini-IRCap v2.4
Cliente GUI de IRC en español para Windows, inspirado en el script IRcap para mIRC. Desarrollado en VB.NET utilizando la librería SmartIRC4net de Meebey.

Aplicación ligera y ejecutable de 348 KB (no requiere instalación). Compatible con cualquier red de IRC, con soporte para comandos y conexión cifrada SSL. Incluye manejo de eventos básicos, opciones de Operador (KICK, BAN, OP, DEOP, INVITE, etc., en menú), decodificación de texto decorativo, tamaño de texto ajustable y cuatro colores disponibles. Simple, funcional y práctica.

Nota: La librería SmartIRC4net (.dll) se integró junto al ejecutable utilizando ILRepack, lo que podría generar un falso positivo y/o el bloqueo por parte de algunos antivirus o navegadores. Para evitar este inconveniente, también se ofrece una versión comprimida en el archivo "Mini-IRCap.2.4.rar", que incluye la dll por separado.
